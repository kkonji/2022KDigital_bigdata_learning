# K-Digital BigData Learning
## Project3: Corona
- Bigdata analysis 강의 때 배운 내용을 바탕으로 corona 에 관해 EDA 진행

### 맡은 역할
- 맡은 부분 : corona.ipynb
- Dataset: 
    - 한국관광 데이터랩
- Used Package :
    - pandas, numpy, matplotlib, folium
### EDA 작업
- 주요 관광명소들의 인기변화
- 2021년 기준 '한국관광 데이터랩' 사이트에서 관광명소 자료를 받아와서, 구글트렌드에서 인기변화를 살펴본다.
